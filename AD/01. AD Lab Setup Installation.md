## Simple Architecture 

* DC01 = Domain Controller (DNS + AD)
* WC01 = Dual Network (External + Internal)
* WC02 = Internal Network Only
* Sab machines DC (192.168.2.100) ko DNS use kar rahi hain

Ye pura setup AD Pentesting practice ke liye banaya gaya hai:

* Enumeration

* Privilege Escalation
* Kerberos Attacks
* Pass-the-Hash
