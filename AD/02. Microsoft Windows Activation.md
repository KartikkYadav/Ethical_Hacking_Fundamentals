
# Microsoft-Windows-Activation

## Microsoft Windows Activation

### Microsoft Activation Scripts
You can use Microsoft Activation Scripts to activate Windows 11 and Office products. This script is available on GitHub and is widely used for easy and automated activation.

**Repository Link:**
[Microsoft Activation Scripts - https://github.com/massgravel/Microsoft-Activation-Scripts](https://github.com/massgravel/Microsoft-Activation-Scripts)

### Method 1 - PowerShell Activation

1. **Open PowerShell:**
   * Click the Start Menu, type "PowerShell," and open it.

2. **Run the appropriate command based on your Windows version:**
   * For **Windows 8, 10, 11**:

   ```powershell
   irm https://get.activated.win | iex
   ```

* If access is blocked (by ISP/DNS), try:

   ```powershell
   iex (curl.exe -s --doh-url https://1.1.1.1/dns-query https://get.activated.win | Out-String)
   ```

* For **Windows 7** and later:

   ```powershell
   iex ((New-Object Net.WebClient).DownloadString('[https://get.activated.win](https://get.activated.win)'))
   ```

3. **Follow the on-screen activation menu** that appears, selecting the highlighted options to activate Windows or Office.
4. **Completion:** After following the prompts, Windows or Office should be activated.
