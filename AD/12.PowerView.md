# 📌 Overview

PowerView is a PowerShell tool used for:

- Enumerating Active Directory (AD)
- Gathering domain information

- Privilege escalation mapping

- Users
- Groups
- Computers
- Domain Controllers
- Trust Relationships
- Sessions
- Permissions
- GPOs (Group Policy Objects)

# 🌐 Domain Enumeration

## Get Domain Info

    Get-Domain
    Get-NetDomain

---

## Get Domain SID

    Get-DomainSID

---

## Get Domain Controllers

    Get-DomainController
    Get-NetDomainController

---

## Specific Domain

    Get-Domain -Domain infosecwarrior.local

---

# 👤 User Enumeration

## List All Users

    Get-DomainUser

---

## Show Only Usernames

    Get-DomainUser | select samaccountname

---

## User Logon Info

    Get-DomainUser -Properties lastlogondate

---

## Find Specific User

    Get-DomainUser -Identity username

---

# 👥 Group Enumeration

## List Groups

    Get-DomainGroup

---

## Group Members

    Get-DomainGroupMember -Identity "Domain Admins"

---

## Find Admin Groups

    Get-DomainGroup | findstr admin

---

# 🖥️ Computer Enumeration

## List Computers

    Get-DomainComputer

---

## Find Specific Computer

    Get-DomainComputer -Identity PC01

---

# 🔐 Privilege & Permission Enumeration

## Find Local Admin Access

    Find-LocalAdminAccess

## Find Logged-in Users

## Find Domain Admins
