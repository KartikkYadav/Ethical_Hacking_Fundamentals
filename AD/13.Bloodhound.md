# BloodHound – Active Directory Enumeration

## BloodHound

BloodHound is an **Active Directory (AD) enumeration tool** that uses **graph theory** to discover relationships between AD objects and identify **privilege escalation paths**.

It helps attackers and defenders visualize how:

- Permissions
- Group memberships
- ACLs

can lead to **Domain Admin compromise**.

---

# Key Features

- Graph-based visualization of AD relationships
- Identifies privilege escalation paths
- Detects misconfigurations in AD environments
- Uses **Neo4j** graph database
- Supports **custom Cypher queries**

---

# Objects BloodHound Maps

- Users
- Groups
- Computers
- Sessions
- ACL permissions
- Trust relationships

---

# BloodHound Architecture

    SharpHound / BloodHound.py
            ↓
      Collect AD Data
            ↓
      JSON / ZIP Files
            ↓
        Neo4j Database
            ↓
      BloodHound GUI
