```bash
# Unpack rockyou (Debian/Ubuntu path)

sudo gunzip /usr/share/wordlists/rockyou.txt.gz

cat /usr/share/wordlists/rockyou.txt

# View compressed contents without extracting

zcat /usr/share/wordlists/rockyou.txt.gz

             
# Copy compressed file to another location and extract                                     
cp -v /usr/share/wordlists/rockyou.txt.gz /opt/rockyou.txt.gz 
 
cd /opt

gunzip rockyou.txt.gz

# Search for a literal word in a list

grep "^ilovex" rockyou.txt

```

# What is SecLists

**SecLists** is a curated collection of multiple types of wordlists used in security testing: usernames, passwords, URLs, fuzzing payloads, web-content lists, discovery lists, and more. Widely used with tools like `gobuster`, `wfuzz`, `dirb`, `hydra`, `john`, etc.

---

# Install

**On Debian / Ubuntu / Kali:**

```bash
sudo apt update
sudo apt install seclists

```

**From GitHub (latest):**

```bash
git clone https://github.com/danielmiessler/SecLists.git ~/SecLists

```

---

# Common locations

- If installed via APT:
    
    /usr/share/seclists/Passwords
    
- If cloned from GitHub:
    
    `~/SecLists/`
    

To list package files (APT install):

```bash
dpkg -L seclists

```

---

# Main categories you’ll see

- `Discovery/` — web content, directories, common files (e.g. `Web-Content`, `DNS`)
- `Passwords/` — password lists (various sizes and sources)
- `Usernames/` — username lists and common user patterns
- `Payloads/` — fuzzing payloads, injection strings
- `Fuzzing/` — input strings for fuzzers
- `Web-Content/` — large lists for directory brute-forcing
- `Credentials/` — common credential pairs / combos

---

# Useful example lists (common choices)

- `/usr/share/seclists/Discovery/Web-Content/common.txt` — small common web paths
- `/usr/share/seclists/Discovery/Web-Content/directory-list-2.3-medium.txt` — medium sized dir list
- `/usr/share/seclists/Passwords/Leaked-Databases/rockyou.txt` — well-known password list (if present)

# Example usage commands

Search for a string in lists:

```bash
grep -i -n 'admin' /usr/share/seclists/Usernames/* | less

```

Use with `gobuster`:

```bash
gobuster dir -u https://example.com -w /usr/share/seclists/Discovery/Web-Content/common.txt -t 50

```

Use with `ffuf` (fast web fuzzer):

```bash
ffuf -u https://example.com/FUZZ -w /usr/share/seclists/Discovery/Web-Content/common.txt -fs 0

```

Use with `hydra` for password guessing:

```bash
hydra -l admin -P /usr/share/seclists/Passwords/Leaked-Databases/rockyou.txt ftp://target

```

Use with `john` (format properly first):

```bash
john --wordlist=/usr/share/seclists/Passwords/Leaked-Databases/rockyou.txt --rules myhashes.txt

```

Quick peek at top entries:

```bash
head -n 40 /usr/share/seclists/Discovery/Web-Content/common.txt

```

Count words / lines:

```bash
wc -l /usr/share/seclists/Discovery/Web-Content/common.txt

```

Preview compressed lists without extracting:

```bash
zcat file.txt.gz | head

```
