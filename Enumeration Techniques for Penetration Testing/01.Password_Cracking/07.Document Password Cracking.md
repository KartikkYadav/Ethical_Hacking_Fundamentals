### Hashcat Command Explanation

The text also explains the components of a typical dictionary attack command:

- **`a 0`**: Specifies the **dictionary attack mode**.
- **`-username`**: Used to handle the output format correctly.
- **`o cracked.txt`**: Outputs the cracked passwords to the specified file.
- **`office-hash-list.txt`**: Is your file with the Office hashes.
- **`rockyou.txt`**: Is your password candidate list (the dictionary file).

# Cracking-SSH-Private-Keys-with-Hashcat

## Cracking SSH Private Keys with Hashcat

SSH private keys in the `$sshng$` format can be cracked using Hashcat with the appropriate mode based on the key type indicated in the hash.

```bash
ssh-keygen -b 2048 -t rsa -f id_rsa -q -N "armour123"

  ssh2john id_rsa > ssh_id_rsa.hash.txt
  
  vim ssh_id_rsa.hash.txt 
  
  
  hashcat -hh | grep -i "rsa"

```

## Hashcat SSH Key Modes

| Type Value | Hashcat Mode | Key Type / Cipher |
| --- | --- | --- |
| 0 | 22900 | RSA/DSA + 3DES |
| 1 | 22931 | RSA, DSA (AES-128) |
| 2 | 22932 | RSA/DSA/EC + Bcrypt PBKDF + AES-256-CBC |
| 3 | 22933 | EC (AES-128) |
| 4 | 22934 | RSA, DSA (AES-192) |
| 5 | 22935 | RSA, DSA (AES-256) |
| 6 | 22936 | RSA/DSA/EC + Bcrypt PBKDF + AES-256-CTR |

## Workflow Example

- For a hash starting with `$sshn g$1$` (type value 1), use mode **22931**:

```bash
hashcat -m 22931 -a 0 id_rsa_armour1.hash /opt/darkweb2017-top1000.txt

```

- For a hash starting with `$sshn g$6$` (type value 6), use mode **22936**:

```bash
hashcat -m 22936 -a 0 id_rsa_armour2.hash /opt/darkweb2017-top1000.txt

```
