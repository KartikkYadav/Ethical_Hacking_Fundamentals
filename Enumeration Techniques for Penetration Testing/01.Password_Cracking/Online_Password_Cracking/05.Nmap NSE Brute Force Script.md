# 🔵 **Nmap NSE Brute Force Script**

```
nmap -v -sT -p <port> --script=<scriptname> --script-args userdb=<userfile>,passdb=<passfile> <target>

```

- **v** : verbose output
- **sT** : TCP connect scan
- **p** : port number
- **-script** : specify NSE script
- **-script-args** : pass username and password list files
- : target IP or hostname

---

# 🌐 **HTTP Brute Force on Port 80**

```
nmap -v -sT -p 80 --script=http-brute.nse --script-args userdb=/opt/username.txt,passdb=/opt/password.txt 192.168.1.42

```

or with quoted script arguments:

```
nmap -v -sT -p 80 --script=http-brute.nse --script-args="userdb=/opt/username.txt,passdb=/opt/password.txt" 192.168.1.42

```
