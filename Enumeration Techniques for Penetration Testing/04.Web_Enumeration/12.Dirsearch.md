# dirb

`dirb` is a command-line web content scanner that brute forces directories using a wordlist.

## Basic Usage

    dirb http://192.168.1.211

## Custom Wordlist Example

    dirb http://192.168.1.211 /usr/share/seclists/Discovery/Web-Content/directory-list-2.3-medium.txt

---

# dirbuster

`dirbuster` is a GUI-based directory brute force tool created by OWASP.

## Start Dirbuster

    dirbuster

Steps:

1. Enter target URL  
2. Select a wordlist  
3. Configure scanning options  
4. Click **Start Attack**

# Dirsearch – Directory Brute Forcing

Dirsearch is a Python-based directory brute force tool.

Repository:

https://github.com/maurosoria/dirsearch

---

## Installation

Install via APT:

    apt install dirsearch

Manual install:

    cd /opt
    git clone https://github.com/maurosoria/dirsearch.git
    cd dirsearch
    pip3 install -r requirements.txt

---

## Basic Usage

Scan a target:

    dirsearch -u http://192.168.1.211

Increase threads:

    dirsearch -u http://192.168.1.211 -t 100

Use custom wordlist:

    dirsearch -u http://192.168.1.211 -w /usr/share/seclists/Discovery/Web-Content/raft-medium-files.txt

---

## Recursive Scanning

Perform recursive scan:

    dirsearch -u http://192.168.1.31 -r -t 100

Limit recursion depth:

    dirsearch -u http://192.168.1.31 -r -R 3 -t 100

---

## Proxy Support

Send traffic through Burp Suite:

    dirsearch -u http://192.168.1.30/dvwa/ --proxy=http://127.0.0.1:8080

---

## Extension Based Scanning

Scan PHP files:

    dirsearch -u http://192.168.1.31 -e php

Scan multiple extensions:

    dirsearch -u http://192.168.1.31 -e php,sql

---

## Save Output

Save results to file:

    dirsearch -u http://192.168.1.31 -o result.txt
