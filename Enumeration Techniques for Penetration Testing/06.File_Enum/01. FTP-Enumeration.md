# FTP-Enumeration

## FTP Enumeration

FTP (File Transfer Protocol) enumeration is the process of discovering information about an FTP service such as:

- Service version  
- User access (anonymous or authenticated)  
- File structure  
- Permissions  
- Potential vulnerabilities  

This is a key step during reconnaissance in penetration testing.

---

## 1. Banner Grabbing

Identify the FTP service and version.

### Using Netcat

```
nc <target_ip> 21
nc -v 192.168.1.207 21
nc -v 192.168.1.207 2121
```

---

### Using Telnet

```
telnet <target_ip> 21
telnet 192.168.1.207 21
telnet 192.168.1.207 2121
```

---

### What to Look For

- FTP banner (e.g., vsftpd 3.0.3)  
- OS hints  
- Misconfigurations  

---

## 2. FTP Version Detection

### Nmap Scans

```
nmap -v -sV -sT -sC -A -p 21,2121 192.168.1.207
```

---

### Scan FTP Ports Using Wildcard

```
nmap -v -p ftp* -sT -sV -A 192.168.1.207
```

---

### Run Specific NSE Script

```
nmap -v -sT -p 21 --script=ftp-syst.nse 192.168.1.207
```

---

### Run All FTP Scripts

```
nmap -v -sT -sV -A -O -p 21,2121 --script=ftp* 192.168.1.250
```

---

## 3. Anonymous Login Check

### Common Credentials

| Username  | Password              |
|----------|----------------------|
| anonymous| anonymous@domain.com |
| ftp      | ftp@domain.com       |
| guest    | guest@domain.com     |

---

### Manual Login

```
ftp <target_ip>
```

**Example:**
```
Name: anonymous
Password: anonymous@domain.com
```

✔ Success Indicator:
```
230 Login successful
```

---

### Using Nmap

```
nmap -p21 --script=ftp-anon <target_ip>
```

---

### Download Files

```
wget -m ftp://anonymous@192.168.1.207
wget -m --no-passive ftp://anonymous@192.168.1.207
```

---

### File Analysis

```
file filename
strings filename
cat filename
```

---

## 4. Directory Traversal

Attempt to access restricted directories.

### Key Commands

```
ls -al
pwd
cd .
cd /etc
cd ../../../../etc
```

---

### Indicators of Vulnerability

- Access outside FTP root  
- Access to sensitive files (/etc/passwd)  

---

## 5. FTP Commands Reference

| Command | Description              |
|--------|--------------------------|
| ls     | List files               |
| cd     | Change directory         |
| pwd    | Show current directory   |
| get    | Download file            |
| put    | Upload file              |
| delete | Remove file              |
| mget   | Download multiple files  |
| mput   | Upload multiple files    |

## 8. Writable Directory Check

### Test Write Permissions

```
mkdir test
cd test
put test.txt
delete test.txt
rmdir test
```

---

### Indicators

- File upload allowed  
- Directory creation allowed

## 9. Brute-Forcing FTP Credentials

### Hydra

```
hydra -l user -P passwords.txt ftp://192.168.1.207
hydra -L users.txt -P passwords.txt ftp://192.168.1.207
hydra -C creds.txt ftp://192.168.1.207
```

---

### Medusa

```
medusa -h 192.168.1.207 -u user -P passwords.txt -M ftp
```

---

### Ncrack

```
ncrack -p 21 -U users.txt -P passwords.txt 192.168.1.207
```

---

## 12. Common Vulnerabilities

- Anonymous login enabled  
- Weak credentials  
- Writable directories  
- FTP bounce attack  
- Outdated FTP software  
- Clear-text communication  

---

## 13. Mitigation Best Practices

- Disable anonymous access  
- Use strong passwords  
- Enable account lockout  
- Restrict access by IP  
- Use FTPS/SFTP instead of FTP  
- Monitor logs regularly
