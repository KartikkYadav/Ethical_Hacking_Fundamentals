## TCP Connect Scan

- Load TCP scan module


      use auxiliary/scanner/portscan/tcp

- View options

      msf6 auxiliary(scanner/portscan/tcp) > show options

- Set target host

      msf6 auxiliary(scanner/portscan/tcp) > set RHOSTS 192.168.1.152

- Scan all ports

      msf6 auxiliary(scanner/portscan/tcp) > set PORTS 1-65535

- High number of threads (Linux/Cygwin)

      msf6 auxiliary(scanner/portscan/tcp) > set THREADS 200

- Run scanner

      msf6 auxiliary(scanner/portscan/tcp) > run

### SYN Port Scan

   Fast, stealthy, and widely used.

-  Start Metasploit

        msfconsole -q

-  Search available portscan modules

        msf6 > search portscan

-  Load SYN scan module

        msf6 > use auxiliary/scanner/portscan/syn

- Show available options

      msf6 auxiliary(scanner/portscan/syn) > show options

-  Set target host(s)

        msf6 auxiliary(scanner/portscan/syn) > set RHOSTS 192.168.1.31

- Scan all ports

      msf6 auxiliary(scanner/portscan/syn) > set PORTS 1-65535

- Increase speed (Linux recommended)

      msf6 auxiliary(scanner/portscan/syn) > set THREADS 50

- Run the scan

      msf6 auxiliary(scanner/portscan/syn) > run
