# 🔹 RPC Enumeration

**Remote Procedure Call (RPC)** is a protocol that allows a program to request services from another system across a network.

RPC enumeration is commonly used during penetration testing to identify:

- Users
- Groups
- Shares
- Policies
- RPC interfaces
- Services

It is especially useful in:

- Windows Active Directory environments
- SMB-based networks
- Legacy Unix RPC services

# 🛠️ Tools for RPC Enumeration

# 🟣 1️⃣ rpcclient

`rpcclient` is a Linux tool used to communicate with Windows SMB/RPC services.

---

## Anonymous Connection

```bash
rpcclient -U "" <target-ip>
```

Example:

```bash
rpcclient -U "" 192.168.10.145
```

---

# 🟢 Common rpcclient Commands

## Enumerate Domain Users

```bash
enumdomusers
```

---

## Query Specific User

```bash
queryuser <RID>
```

Example:

```bash
queryuser 0x42a
```

---

## Enumerate Domain Groups

```bash
enumdomgroups
```

---

## Query Group Information

```bash
querygroup <RID>
```

---

## Get Domain Password Policy

```bash
getdompwinfo
```

---

## LSA Information


# 🟣 2️⃣ Nmap RPC & SMB Scripts

## RPC Info Scan (Port 111)

```bash
nmap -sT -p 111 --script=rpcinfo <target-ip>
```

Example:

```bash
nmap -sT -p 111 --script=rpcinfo 192.168.10.145
```

---

## RPC Info Scan (Port 135)

```bash
nmap -sT -p 135 --script=rpcinfo <target-ip>
```

---

# 🟢 SMB Enumeration Scripts

```bash
nmap -p 445 --script=smb-enum-users,smb-enum-shares <target-ip>
```

Example:

```bash
nmap -p 445 --script=smb-enum-users,smb-enum-shares 192.168.10.145
```

# 🔍 Common RPC Ports

| Port | Service |
|---|---|
| `111` | RPCbind / Portmapper |
| `135` | Microsoft RPC Endpoint Mapper |
| `139` | NetBIOS Session Service |
| `445` | SMB over TCP |

# ⚠️ Common Security Issues

- Anonymous SMB/RPC access
- Weak password policies
- Exposed shares
- Excessive user enumeration
- Legacy SMB configurations
- Misconfigured Windows services

---

# 🛡️ Defense & Hardening

## Recommended Security Measures

- Disable unnecessary RPC services
- Restrict access to:
  - Port 135
  - SMB services
- Enable Windows Firewall
- Use SMB signing
- Enforce strong passwords
- Disable anonymous enumeration
- Patch legacy SMB vulnerabilities

---
