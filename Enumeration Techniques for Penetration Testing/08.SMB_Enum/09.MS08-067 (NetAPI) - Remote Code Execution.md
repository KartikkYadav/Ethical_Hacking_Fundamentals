# MS08-067 (NetAPI) - Remote Code Execution

MS08-067 is a critical remote code execution vulnerability in the Windows Server service.

It is exploited over SMB (`TCP Port 445`).

Commonly used in penetration testing labs and legacy Windows environments.

# 🔹 Affected Operating Systems

- Windows XP
- Windows Server 2003
- Older unpatched Windows systems

---

# 🔹 Common Indicators

- SMB service exposed on port `445`
- Legacy Windows operating systems
- Missing MS08-067 security patch
- Vulnerable NetAPI service

---

# 🔹 Security Recommendations

- Apply Microsoft security patches immediately.
- Disable SMBv1 where possible.
- Restrict SMB access using firewall rules.
- Segment legacy systems from production networks.
- Monitor SMB traffic for exploitation attempts.
