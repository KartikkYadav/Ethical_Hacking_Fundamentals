# 🛠 SMB-Enumeration-using-netexec

## 📌 SMB Enumeration using netexec (formerly CrackMapExec)

---

# ✅ Prerequisites

- `netexec` installed
    
    (`https://github.com/Pennyw0rth/NetExec`)
    
- Target system reachable
    
    (e.g., `192.168.1.100`)
    
- Optional: Valid SMB credentials

---

# 🌐 1. Basic SMB Info

```bash
netexec smb 192.168.1.100
```

```bash
netexec smb ip-list.txt
```

```bash
netexec smb smb-hosts.txt
```

```bash
netexec smb ip-list.txt --log netexec-smb-host.txt
```

## 📌 Shows

- OS version
- Hostname
- Domain/workgroup
- SMB signing status
- SMBv1 status

---

# 👥 2. Null Session / Guest Access Test

```bash
netexec smb 192.168.10.52 -u '' -p '' --shares
```

```bash
netexec smb ip-list.txt -u '' -p '' --shares --log netexec-smb-shares.txt
```

---

# 👤 3. Enumerate Users (Authenticated)

```bash
netexec smb 192.168.10.52 -u <username> -p <password> --users
```

## 📂 Using credential files

```bash
netexec smb 192.168.10.52 -u users.txt -p passwords.txt --users
```

```bash
netexec smb smb-hosts.txt -u '' -p '' --users
```

---

# 📁 4. List SMB Shares

```bash
netexec smb 192.168.10.52 -u <username> -p <password> --shares
```

```bash
netexec smb smb-hosts.txt -u '' -p '' --shares
```

## 📌 Shows

- Accessible shares
- Permissions

---

# 🔓 5. Bruteforce SMB Logins

```bash
netexec smb 192.168.1.0/24 -u users.txt -p passwords.txt
```

---

# 🔐 6. Password Policy Enumeration

```bash
netexec smb 192.168.10.52 -u <username> -p <password> --pass-pol
```

---

# 🧾 7. Dump SAM Hashes (if privileges allow)

```bash
netexec smb 192.168.10.52 -u <username> -p <password> --sam
```

```bash
netexec smb 192.168.10.52
```

## 👥 Step 2: Guest access check

```bash
netexec smb 192.168.10.52 -u '' -p '' --shares
```

```bash
netexec smb 192.168.10.52 -u admin -p 'Pa$$w0rd123!' --shares --users --pass-pol
```

# 📝 Notes

- `netexec` is modular, fast, and designed for AD environments
- Supports NTLM, Kerberos, and credential spraying
- Useful for large-scale enumeration and lateral movement
- Always check SMB signing status (important for relay attacks)
