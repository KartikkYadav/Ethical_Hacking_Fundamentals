# 📘 MySQL Enumeration Queries Reference

A reference collection of commonly used MySQL queries for database enumeration, privilege inspection, user auditing, process monitoring, server configuration, storage analysis, and security checks.

---

# 🟢 1. Database & Table Operations

## 🟩 List All Databases

```sql
SHOW DATABASES;
```

---

## 🟩 Select a Database

```sql
USE Webapp;
```

---

## 🟩 List Tables

```sql
SHOW TABLES;
```

---

## 🟩 Show Table Columns

```sql
SHOW COLUMNS FROM mysql.user;
```

or

```sql
DESCRIBE mysql.user;
```

---

## 🟩 Show Table Structure

```sql
SHOW CREATE TABLE mysql.user;
```

---

## 🟩 Count Records

```sql
SELECT COUNT(*)
FROM users;
```

---

## 🟩 Dump Table Contents

```sql
SELECT *
FROM users;
```

---

## 🟩 Display Limited Records

```sql
SELECT *
FROM users
LIMIT 10;
```

# 🟢 5. Database Size & Storage

## 🟩 Database Sizes

```sql
SELECT
    table_schema AS "Database Name",
    ROUND(SUM(data_length + index_length) / 1024 / 1024, 2) AS "Size (MB)"
FROM information_schema.tables
GROUP BY table_schema;
```

---

## 🟩 Largest Tables

```sql
SELECT
    table_schema,
    table_name,
    ROUND((data_length + index_length) / 1024 / 1024, 2) AS "Size (MB)"
FROM information_schema.tables
ORDER BY (data_length + index_length) DESC;
```

# 🟢 11. Quick Enumeration Workflow

```sql
-- List databases
SHOW DATABASES;

-- Select target database
USE target_db;

-- List tables
SHOW TABLES;

-- View table structure
DESCRIBE users;

-- Dump sample data
SELECT * FROM users LIMIT 10;

-- Check current user
SELECT CURRENT_USER();

-- View privileges
SHOW GRANTS;

-- Check server version
SELECT @@version;

-- View active sessions
SHOW PROCESSLIST;
```

# 🟢 12. Commonly Used Queries

| Query | Purpose |
|--------|---------|
| `SHOW DATABASES;` | List all databases |
| `SHOW TABLES;` | List tables in current database |
| `DESCRIBE table;` | Display table structure |
| `SELECT * FROM table;` | Dump table contents |
| `SELECT VERSION();` | Show MySQL version |
| `SELECT USER();` | Show connected user |
| `SELECT CURRENT_USER();` | Show authenticated user |
| `SHOW GRANTS;` | Display privileges |
| `SHOW PROCESSLIST;` | View active sessions |
| `SHOW VARIABLES;` | Display server configuration |
| `SHOW STATUS;` | Display server status |
| `SELECT @@hostname;` | Show server hostname |
| `SELECT @@datadir;` | Show data directory |
| `SHOW VARIABLES LIKE 'secure_file_priv';` | Check file operation restrictions |
