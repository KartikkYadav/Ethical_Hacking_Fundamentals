# 游닂 Microsoft SQL Server Client Tools & Administration

## 游닀 Overview

Microsoft SQL Server provides several client utilities for administration, auditing, database management, and authorized security assessments. This reference covers **Impacket MSSQL Client**, **SQLCMD**, common enumeration queries, and related administration tools.

---

# 游릭 1. Impacket MSSQL Client (Fortra Impacket)

`impacket-mssqlclient` is a command-line MSSQL client commonly used by administrators, incident responders, and security professionals to connect to Microsoft SQL Server using SQL Server or Windows authentication.

---

## 游릴 SQL Authentication

### Username Prompt

```bash
impacket-mssqlclient sa@192.168.1.52
```

### Username & Password

```bash
impacket-mssqlclient sa:Password123@192.168.1.52
```

---

## 游릴 Windows Authentication

```bash
impacket-mssqlclient DOMAIN/username@192.168.1.52 -windows-auth
```

```bash
impacket-mssqlclient NS2/administrator@192.168.1.52 -windows-auth
```

```bash
impacket-mssqlclient PublicUser:Password123@192.168.1.52 -windows-auth
```

---

## 游릴 Kerberos Authentication

```bash
impacket-mssqlclient DOMAIN/user@sqlserver.domain.local -k
```

---

## 游릴 Pass-the-Hash Authentication

```bash
impacket-mssqlclient administrator@192.168.1.52 \
-hashes LMHASH:NTHASH \
-windows-auth
```

---

## 游릴 Help

```bash
impacket-mssqlclient -h
```

---

## 游릴 Common Options

| Option          | Description                  |
| --------------- | ---------------------------- |
| `-windows-auth` | Windows Authentication       |
| `-k`            | Kerberos Authentication      |
| `-hashes`       | Pass-the-Hash Authentication |
| `-dc-ip`        | Domain Controller IP         |
| `-target-ip`    | Target IP Address            |
| `-file`         | Execute commands from file   |
| `-debug`        | Enable debug output          |

# 游릭 2. Common Enumeration Queries

## 游릴 Server Information

```sql
SELECT @@VERSION;

SELECT @@SERVERNAME;

SELECT SERVERPROPERTY('Edition');

SELECT SERVERPROPERTY('ProductVersion');
```

---

## 游릴 Current User & Roles

```sql
SELECT SYSTEM_USER;

SELECT CURRENT_USER;

SELECT USER_NAME();

SELECT IS_SRVROLEMEMBER('sysadmin');

SELECT IS_SRVROLEMEMBER('securityadmin');

SELECT IS_SRVROLEMEMBER('serveradmin');
```

---

## 游릴 Server Roles

```sql
SELECT name
FROM sys.server_principals
WHERE type_desc='SERVER_ROLE';
```

---

## 游릴 SQL Logins

```sql
SELECT name
FROM sys.sql_logins;
```

```sql
SELECT name, type_desc
FROM sys.server_principals;
```

---

## 游릴 Databases

```sql
SELECT name
FROM sys.databases;
```

```sql
SELECT name, state_desc
FROM sys.databases;
```

---

## 游릴 Current Database

```sql
SELECT DB_NAME();
```

---

## 游릴 Database Users

```sql
SELECT name
FROM sys.database_principals
WHERE type IN ('S','U','G');
```

---

## 游릴 Tables

```sql

       TABLE_NAME

```

---

## 游릴 Columns

```sql

       COLUMN_NAME

```

---


# 游릭 3. SQLCMD

Microsoft SQLCMD is the official Microsoft command-line utility for SQL Server administration.

---

## 游릴 SQL Authentication

```bash
sqlcmd -S 192.168.1.52 -U sa -P Password123
```

---

## 游릴 Specify Port

```bash
sqlcmd -S 192.168.1.52,1433 -U sa -P Password123
```

---

## 游릴 Connect to a Database

```bash
sqlcmd -S 192.168.1.52 \
-U sa \
-P Password123 \
-d master
```

---

## 游릴 Windows Authentication

```bash
sqlcmd -S SQLSERVER01 -E
```

---

## 游릴 Execute a Query

```bash
sqlcmd -S 192.168.1.52 \
-U sa \
-P Password123 \
-Q "SELECT @@VERSION"
```

---

## 游릴 Execute a SQL Script

```bash
sqlcmd -S 192.168.1.52 \
-U sa \
-P Password123 \
-i script.sql
```

# 游릭 4. SQL Server Administration Features

SQL Server includes administrative features that may be available depending on server configuration and assigned privileges.

---

## 游릴 Verify `xp_cmdshell` Status

```sql
SELECT value_in_use
FROM sys.configurations
WHERE name='xp_cmdshell';
```

> **Note:** `xp_cmdshell` is disabled by default on modern SQL Server installations. Enabling or using it should only be done by authorized administrators in accordance with organizational security policies.

---

## 游릴 Security Considerations

* `xp_cmdshell` runs commands under the SQL Server service account.
* Over-privileged service accounts increase risk.
* Microsoft recommends disabling `xp_cmdshell` unless operationally required.
* Restrict access to trusted administrators.
* Monitor and audit usage regularly.

# 游릭 6. Install SQLCMD (Linux)

## 游릴 Add Microsoft Repository

```bash
curl https://packages.microsoft.com/keys/microsoft.asc | sudo apt-key add -

curl https://packages.microsoft.com/config/ubuntu/22.04/prod.list \
| sudo tee /etc/apt/sources.list.d/mssql-release.list
```

---

## 游릴 Install SQLCMD

```bash
sudo apt update

sudo ACCEPT_EULA=Y apt install -y mssql-tools18 unixodbc-dev
```

---

## 游릴 Create Symlink

```bash
sudo ln -sf /opt/mssql-tools18/bin/sqlcmd /usr/local/bin/sqlcmd
```

---

## 游릴 Verify Installation

```bash
sqlcmd -?
```

or

```bash
which sqlcmd
```

Expected output:

```text
/usr/local/bin/sqlcmd
```
