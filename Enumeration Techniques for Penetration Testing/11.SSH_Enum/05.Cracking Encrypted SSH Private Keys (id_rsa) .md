## Overview

SSH private key crack karna penetration testing aur security research mein ek common step hai, particularly jab tumhare paas ek protected key (jaise `id_rsa`) ho lekin passphrase na pata ho. Main process mein key ko suitable hash format mein convert karna aur phir John the Ripper ya Hashcat se passphrase brute-force karna shaamil hai.

> **Important:** Yeh actions sirf un systems, accounts, aur keys pe perform karo jinke liye explicit authorization ho.
> 

---

## Step 1 — Key Format Identify Karo

Hash extract karne se pehle key type aur format determine karo:

```bash
file id_rsa
ssh-keygen -l -f id_rsa
```

Modern OpenSSH keys typically `openssh-key-v1` format use karti hain aur bcrypt-based key derivation use karti hain, jabki older keys PEM encoding aur different encryption methods use karti hain.

---

## Step 2 — ssh2john.py Locate Karo

John the Ripper Jumbo mein `ssh2john.py` utility hoti hai jo encrypted SSH private keys ko crackable hash format mein convert karti hai.

```bash
# Script dhundo
locate ssh2john.py
```

Common locations:

```
/usr/share/john/ssh2john.py
/opt/john/run/ssh2john.py
```

Script unavailable ho toh download karo:

```bash
wget https://raw.githubusercontent.com/openwall/john/bleeding-jumbo/run/ssh2john.py
```

---

## Step 3 — SSH Private Key Ko Hash Mein Convert Karo

```bash
# Basic conversion
python ssh2john.py id_rsa > id_rsa.hash

# Full path se
python /usr/share/john/ssh2john.py id_rsa > id_rsa.hash
```

Where:

- `id_rsa` = encrypted SSH private key
- `id_rsa.hash` = extracted hash for password auditing

Output verify karo:

```bash
cat id_rsa.hash
```

Successful extraction ek hash produce karta hai jo John the Ripper aur Hashcat ke recognized format mein hota hai.

---

## Step 4 — Hash Format Types Samjho

Output mein certain hash prefixes specific meanings rakhte hain:

```
$sshng$0$ → SSH key using 3DES
$sshng$1$ → RSA/DSA with AES-128
$sshng$2$ → bcrypt PBKDF + AES-256-CBC/CTR
$sshng$3$ → EC keys with AES-128
$sshng$4$ → RSA/DSA using AES-192
$sshng$5$ → RSA/DSA using AES-256
$sshng$6$ → Typically RSA/DSA/EC + bcrypt_pbkdf + AES-256-CTR
```

Hash file inspect karo:

```bash
cat id_rsa.hash
```

## Step 5 — John the Ripper Se Passphrase Crack Karo

John the Ripper SSH key passphrases ke liye optimized ek efficient tool hai.

**Default Mode:**

```bash
john id_rsa.hash
```

**Custom Wordlist Se:**

```bash
john id_rsa.hash --wordlist=/opt/password.txt
```

**rockyou.txt Se:**

```bash
john id_rsa.hash --wordlist=rockyou.txt
```

**Recovered Passphrases Dikhao:**

```bash
john --show id_rsa.hash
```

**Current Status Display Karo:**

```bash
john --status
```

---

## Step 6 — Hashcat Se Passphrase Crack Karo

Hashcat GPU acceleration leverage kar sakta hai faster testing ke liye.

Shuru karne se pehle correct mode verify karo:

```bash
hashcat --example-hashes | grep -i ssh
```

**OpenSSH Private Key Hashes Ke Liye:**

```bash
hashcat -m 22931 -a 0 id_rsa.hash /opt/password.txt
```

Parameters:

```
-m 22931 = OpenSSH private key hash mode
-a 0     = dictionary attack
```

**Recovered Passphrases Display Karo:**

```bash
hashcat -m 22931 id_rsa.hash --show
```

**Paused Session Resume Karo:**

```bash
hashcat --restore
```

## Troubleshooting

**Unsupported Key Format:**

```bash
# John aur Hashcat update karo
john --list-build-info
hashcat --version
```

**Slow Cracking Speed:**

Modern OpenSSH keys often bcrypt-based key derivation use karti hain jo intentionally computationally expensive hoti hai. Testing speeds traditional password hashes se significantly slower ho sakti hain.
