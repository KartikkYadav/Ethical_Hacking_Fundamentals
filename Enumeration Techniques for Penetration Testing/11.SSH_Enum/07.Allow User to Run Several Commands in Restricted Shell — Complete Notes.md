## Overview

Linux mein ek user account (jaise `user1`) set up kar sakte hain taaki woh sirf selected commands run kar sakein, restricted bash (rbash) aur custom PATH use karke.

---

## Step 1 — New User Banao

```bash
adduser user1
# Ya
useradd user1

# Password set karo
passwd user1
```

---

## Step 2 — Restricted Shell Set Karo

```bash
# /bin/rbash symlink banao (agar exist nahi karta)
ln /bin/bash /bin/rbash

# user1 ka shell restricted bash mein change karo
chsh -s /bin/rbash user1
```

Verify karo:

```bash
grep user1 /etc/passwd
```

Expected output:

```
user1:x:1001:1001::/home/user1:/bin/rbash
```

---

## Step 3 — Restricted Command Directory Banao

```bash
# bin directory banao
mkdir -p /home/user1/bin

# Ownership set karo
chown user1:user1 /home/user1/bin

# Permissions set karo
chmod 755 /home/user1/bin
```

---

## Step 4 — User Ka PATH Configure Karo

Login profile edit karo:

```bash
vim /home/user1/.bash_profile
```

Yeh add karo:

```bash
PATH=$HOME/bin
export PATH
```

Ownership set karo:

```bash
chown user1:user1 /home/user1/.bash_profile
```

---

## Step 5 — Sirf Approved Commands Add Karo

User ko accessible commands ke symlinks banao:

```bash
# ls allow karo
ln -s /bin/ls /home/user1/bin/ls

# cat allow karo
ln -s /bin/cat /home/user1/bin/cat

# echo allow karo
ln -s /bin/echo /home/user1/bin/echo

# id allow karo
ln -s /usr/bin/id /home/user1/bin/id
```

Verify karo:

```bash
ls -l /home/user1/bin
```

Expected output:

```
ls  → /bin/ls
cat → /bin/cat
```

---

## Custom Allowed Commands Array

Additional commands allow karne ke liye `ALLOWED_COMMANDS` array mein add karo:

```bash
ALLOWED_COMMANDS=(
    /bin/ls
    /bin/cat
    /bin/echo
    /usr/bin/id
    /usr/bin/whoami
    /usr/bin/uptime
)
```

Editors (`vim`, `nano`), interpreters (`python`, `perl`, `ruby`), ya programs jo shells spawn kar sakein — inhe kabhi add mat karo.

---

## Step 6 — Startup Files Modification Prevent Karo

```bash
# Ownership restrict karo
chown root:root /home/user1/.bash_profile
chmod 644 /home/user1/.bash_profile

# Optionally immutable banao
chattr +i /home/user1/.bash_profile

# Immutable flag remove karo agar zarurat ho
chattr -i /home/user1/.bash_profile
```

---

## Step 7 — Users Ko Command Directory Modify Karne Se Rokho

Agar user commands add na kar sake:

```bash
chown root:root /home/user1/bin
chmod 755 /home/user1/bin
```

> **Warning:** Agar yeh na karo toh user apne executables add karke restrictions bypass kar sakta hai.
>

## Testing — Restricted Shell Verify Karo

```bash
# Restricted account mein switch karo
su - user1

# Shell check karo
echo $0
# Expected output: -rbash

# Directory change attempt karo
cd /tmp
# Expected output: -rbash: cd: restricted

# Unapproved command run karo
uname
# Expected output: -rbash: uname: command not found

# Approved command run karo
ls
# Expected output: (command executes normally)

# PATH check karo
echo $PATH
ls
cat
id
```

## Security Considerations

`rbash` environment mein yeh programs expose karne se bachao kyunki yeh shell escape functionality provide karte hain:

```
vim     → :set shell=/bin/bash → :shell
vi      → shell escape possible
nano    → shell escape possible
less    → !bash
more    → !bash
man     → shell possible
python  → os.system('bash')
perl    → exec('/bin/bash')
ruby    → exec('/bin/bash')
awk     → system command execution
find    → -exec flag se shell
ssh     → SSH session se bash
scp     → file operations
sftp    → file operations
```

**Example:** `vim` se shell spawn karna:

```
:set shell=/bin/bash
:shell
```

Yeh restriction ka purpose defeat kar deta hai.

## Pentesting Mein rbash Bypass Methods

```bash
# ssh se bypass
ssh user1@TARGET -t bash

# Python se bypass
python -c 'import os; os.system("/bin/bash")'

# vim se bypass
vim → :set shell=/bin/bash → :shell

# awk se bypass
awk 'BEGIN {system("/bin/bash")}'

# find se bypass
find / -name test -exec /bin/bash \;

# less se bypass
less /etc/passwd → !bash
```

---

**Short mein: Restricted Shell = User ko limited commands tak restrict karo, pentesting mein escape techniques use karo!** 💪
