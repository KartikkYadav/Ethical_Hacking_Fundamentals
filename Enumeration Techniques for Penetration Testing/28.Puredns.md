# PureDNS

[PureDNS Repository](https://github.com/d3mondev/puredns)

PureDNS is a powerful and fast domain resolver and subdomain brute-forcing tool.  
It supports advanced features like filtering wildcard domains and avoiding DNS poisoning.  
It works best with **MassDNS** and **DNSValidator**.

---

## Prerequisites

### MassDNS

MassDNS is a high-performance DNS stub resolver required by PureDNS.

#### Clone the repository

    cd /opt
    git clone https://github.com/blechschmidt/massdns.git

### Compile the code

    cd /opt/massdns
    make

### Install the binary

    make install

### Install PureDNS
PureDNS is written in Go.

    go install github.com/d3mondev/puredns/v2@latest

Move the binary to a system-wide path:

    cp -v /root/go/bin/puredns /usr/local/bin

## DNSValidator
DNSValidator is used to generate and validate working public DNS resolvers.

### Download nameserver list

    wget https://public-dns.info/nameservers-all.txt -O /opt/nameservers-all.txt

### Clone DNSValidator

    cd /opt
    git clone https://github.com/vortexau/dnsvalidator.git
    cd dnsvalidator

### Install Python requirements

    pip install -r requirements.txt

### Install DNSValidator

    python3 setup.py install

### Run DNSValidator

    dnsvalidator

### Help menu

    dnsvalidator -h

## Additional Resolver Sources

### Clone Trickest resolvers

    git clone https://github.com/trickest/resolvers.git

### Download extra resolver lists

    wget https://public-dns.info/nameservers.txt
    wget https://raw.githubusercontent.com/danielmiessler/SecLists/master/Miscellaneous/dns-resolvers.txt

### Count resolvers

    cat /usr/share/seclists/Miscellaneous/dns-resolvers.txt | wc -l

## Validate and Output Resolver List

### Using public-dns.info

    dnsvalidator -tL /opt/nameservers-all.txt -threads 50 -o /opt/resolvers.txt

### Using temporary file

    dnsvalidator -tL /tmp/dns-resolvers.txt -threads 50 -o /opt/resolvers.txt

### Using SecLists

    dnsvalidator -tL /usr/share/seclists/Miscellaneous/dns-resolvers.txt -threads 50 -o /opt/resolvers.txt

## Usage

### Basic Commands
Check version:

    puredns -v

Help menu:

    puredns -h

Bruteforce help:

    puredns bruteforce -h

### Bruteforce Mode

    puredns bruteforce /opt/subdomain-wordlists/best-dns-wordlist.txt google.com
