# WordPress Enumeration Notes (Pentesting)

---

# 📌 Overview

WordPress enumeration is used to:
- Identify CMS presence
- Find version, plugins, themes
- Discover users
- Map attack surface

---

# 🔍 1. Nmap HTTP Enumeration

## Command

    nmap -v -p 80,443 --script=http-enum.nse <target>

## Parameters

| Option | Description |
|--------|------------|
| -v | Verbose output |
| -p | Specify ports |
| --script | Run NSE script |

## What it Finds

- Web directories
- CMS paths
- Admin panels
- Login portals
- Config files
- Default pages

---

# 🔍 2. Technology Fingerprinting (WhatWeb)

## Command

    whatweb http://target
    whatweb http://<IP>

## Detects

- CMS type (WordPress)
- CMS version
- Web server (Apache/Nginx)
- Frameworks
- Plugins

## Example Output

    WordPress[6.4.1], Apache[2.4.52], PHP[8.1]

---

# 🔍 3. Vulnerability Scanning (Nikto)

## Command

    nikto -h http://target

## Detects

- Default files
- Backup files
- Exposed admin panels
- Dangerous HTTP methods
- Outdated server versions

---

# 🔍 4. CMS Scanning (Nuclei)

## Command

    nuclei -u http://target

## Advantages

- Fast scanning
- YAML-based templates
- Detects CVEs
- Detects misconfigurations

## Categories

    cves/
    misconfiguration/
    wordpress/
    exposures/

---

# 🔍 5. WordPress Enumeration (Nmap Scripts)

## Plugin & Theme Enumeration

    nmap -v -p 80,443 --script=http-wordpress-enum.nse <target>

## User Enumeration

    nmap -v -p 80,443 --script=http-wordpress-users.nse <target>

## Finds

- Plugins
- Themes
- Version info
- Directory structure
- Users

# 🔍 7. WPScan (Recommended)

## Basic Scan

    wpscan --url http://target

---

## Enumerate Plugins

    wpscan --url http://target --enumerate p

---

## Enumerate Users

    wpscan --url http://target --enumerate u

---

## Enumerate Themes

    wpscan --url http://target --enumerate t

---

## Aggressive Scan

    wpscan --url http://target -e ap,at,u

| Flag | Meaning |
|------|--------|
| ap | All plugins |
| at | All themes |
| u | Users |

---

# 🔍 8. Directory Enumeration

## Gobuster

    gobuster dir -u http://target -w /usr/share/wordlists/dirb/common.txt

## Feroxbuster

    feroxbuster -u http://target

# 📁 Important WordPress Files

| File | Purpose |
|------|--------|
| wp-login.php | Login page |
| xmlrpc.php | Remote API (attack surface) |
| wp-config.php | Database credentials |
| readme.html | Version info |
| license.txt | Version clues |

# 💣 Attack Surface Summary

- XML-RPC → brute force / SSRF
- REST API → user enumeration
- Plugins → RCE / LFI
- Login panel → brute force
- Exposed files → info disclosure

---

# 🧠 Pro Tips

- Always combine findings
- Enumeration = foundation of exploitation
- Focus on:
  - Users
  - Plugins
  - Version
  - Exposed endpoints
