# 🔹 RPC Enumeration Tool - rpcclient

`rpcclient` is a powerful command-line tool used to interact with Windows systems via the SMB protocol.

It is part of the Samba suite and is widely used during:

- SMB enumeration
- RPC enumeration
- Active Directory reconnaissance
- Windows penetration testing

---

# 🔹 What rpcclient Can Enumerate

Using `rpcclient`, testers may enumerate:

- Users
- Groups
- Shares
- Password policies
- Domain information
- SIDs/RIDs
- Sessions
- Open files
- LSA information

---

# 🔹 Anonymous Authentication (Null Session)

If the target allows anonymous access:

```bash
rpcclient -U "" -N <target_IP>
```

---

## Example

```bash
rpcclient -U "" -N 192.168.1.52
```

---

# 🔹 Bulk Check Script for RPC Access

## ✅ Create File

```bash
vim check-rpc.sh
```

---

## ✅ Script Content

```bash
#!/bin/bash

IP_LIST="ips.txt"

while read -r ip; do
    echo "Checking RPC access for IP: $ip"

    rpcclient -U "" -N "$ip" -c "exit" \
    &> "/tmp/rpcclient_check_$ip.log"

    if grep -q "rpcclient" "/tmp/rpcclient_check_$ip.log"; then
        echo "Anonymous RPC login is allowed for $ip"
    else
        echo "No anonymous login for $ip"
    fi

done < "$IP_LIST"
```

---

## ✅ Make Executable

```bash
chmod +x check-rpc.sh
```

---

## ✅ Run Script

```bash
./check-rpc.sh
```

---

## Examples

```bash

```

```bash

```

```bash

```

# 🔍 Common Enumeration Goals

Attackers may attempt to discover:

- Valid usernames
- SMB shares
- Password policies
- Weak credentials
- Administrative access
- Domain information

---

# ⚠️ Common Security Risks

- Null session access
- Weak SMB permissions
- Weak passwords
- Excessive user enumeration
- Misconfigured shares

---

# 🛡 Hardening Recommendations

## Disable Null Sessions

Restrict anonymous SMB/RPC access.

---

## Enable SMB Signing

Helps prevent relay attacks.

---

## Use Strong Password Policies

Enforce:

- Complexity
- Lockouts
- Rotation

---

## Restrict SMB Access

Limit access to trusted hosts.

---

## Disable Legacy SMB Versions

Disable:

```text
SMBv1
```
