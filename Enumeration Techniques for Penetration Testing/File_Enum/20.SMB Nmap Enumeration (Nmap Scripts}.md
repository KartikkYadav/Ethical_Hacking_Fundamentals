
# SMB Nmap Enumeration (Nmap Scripts)

Server Message Block (SMB) is a network file sharing protocol commonly used in Windows environments.

Enumerating SMB services helps identify:

- Users
- Shares
- System details
- Potential security weaknesses

---

# 🔹 Basic SMB Scan

Performs a TCP connect scan with version detection and default scripts on common SMB ports.

```bash
nmap -v -sT -sV -sC -p 135,137,139,445 192.168.2.207
```

---

# 🔹 Targeted SMB NSE Scripts

# ✅ OS Detection

Identifies the operating system and SMB version.

```bash
nmap -v -sT -sV --script=smb-os-discovery.nse -p 139,445 192.168.2.207
```

---

# ✅ System Information

Retrieves detailed host information via SMB.

```bash
nmap -v -sT -sV --script=smb-system-info.nse -p 139,445 192.168.2.207
```

---

# ✅ Security Mode

Checks SMB security configuration such as authentication level and message signing.

```bash
nmap -v -sT -sV --script=smb-security-mode.nse -p 135,137,139,445 192.168.2.207
```

---

# ✅ Share Enumeration

Lists available SMB shares.

```bash
nmap -v -sT -sV --script=smb-enum-shares.nse -p 135,137,139,445 192.168.2.207
```

---

# ✅ Share Browsing

Enumerates shares and lists directory contents.

```bash
nmap -v -sT -sV --script=smb-enum-shares,smb-ls -p 135,137,139,445 192.168.2.207
```

# 🔹 Full SMB Enumeration

```

# 🔹 Authenticated Enumeration

Use valid credentials for deeper enumeration.

```bash
nmap -v -sT -sV --script=smb-enum-shares.nse --script-args smbuser=admin,smbpass='P@ssw0rd' -p 135,137,139,445 192.168.2.207
```

---

# 🔹 Script Discovery and Help

# ✅ List All SMB-Related NSE Scripts

```bash
ls /usr/share/nmap/scripts/smb*
```

---

# ✅ View Help for a Specific Script

```bash
nmap --script-help=smb-enum-users
```

---

# 🔹 Common SMB Ports

| Port | Service |
| --- | --- |
| `135` | RPC |
| `137` | NetBIOS Name Service |
| `139` | NetBIOS Session Service |
| `445` | SMB over TCP |

---

# 🔹 Practical Tips

# ✅ Use `-Pn` if ICMP Is Blocked

```bash
nmap -Pn -p 135,137,139,445 192.168.2.207
```

---

# ✅ Increase Verbosity for Debugging

```bash
nmap -vvv --reason --open -p 135,137,139,445 192.168.2.207
```

---

# 🔹 Additional Notes

- Combine scripts for efficiency during enumeration phases.
- Null sessions may allow enumeration without credentials if SMB is misconfigured.
- SMB enumeration can reveal valuable attack surface information.

---

# 🔹 Summary

SMB Nmap enumeration helps identify:

- Shares
- Users
- Services
- Security configurations
- Domain information

Nmap NSE scripts provide powerful SMB reconnaissance capabilities for penetration testing and security assessments.
