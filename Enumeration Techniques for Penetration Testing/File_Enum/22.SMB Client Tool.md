# SMB Client Tool

---

# 🔹 What is SMB Client Tool?

SMB Client tools are used to interact with **SMB (Server Message Block)** services for:

- Enumeration
- File Access
- Data Transfer
- Remote Share Interaction

👉 These tools are widely used in:

- Penetration Testing
- Red Teaming
- System Administration
- File Sharing Operations

---

# 🟣 1️⃣ smbclient

## 🔹 What is smbclient?

`smbclient` is a command-line tool used to interact with SMB shares similar to an FTP client.

---

# 🔹 Share Enumeration

## ✅ List Available Shares

```bash
smbclient -L //192.168.2.207
```

---

## ✅ List Shares with Authentication

```bash
smbclient -L //192.168.2.207 -U administrator
```

---

## ✅ Attempt Anonymous Access

```bash
smbclient -N -L //192.168.2.207
```

👉 `-N` = No Password (Null Session)

---

# 🟣 Accessing Shares

## ✅ Access a Shared Folder

```bash
smbclient //192.168.2.207/VulnShare
```

---

## ✅ Access Administrative Share (C$)

```bash
smbclient //192.168.2.207/C$ -U administrator
```

---

## ✅ Access ADMIN$ Share

```bash
smbclient //192.168.2.207/ADMIN$ -U administrator
```

---

# 🟣 Authentication Methods

## ✅ Prompt for Password

```bash
smbclient //192.168.2.207/C$ -U administrator
```

👉 Password will be requested interactively.

---

## ✅ Inline Credentials

```bash
smbclient -U 'administrator%Anish@123' //192.168.2.207/C$
```

---

## ✅ List Shares Using Credentials

```bash
smbclient -L 192.168.2.207 -U 'administrator%Anish@123'
```

---

# 🟣 Pass-the-Hash Authentication

## ✅ Authenticate Using NTLM Hash

```bash
smbclient //192.168.2.207/C$ -U administrator --pw-nt-hash ef7fcee5feef61aa827fe4d5e901a516
```

👉 Used during Pass-the-Hash attacks when password is unknown but NTLM hash is available.

---

# 🟣 SMB Protocol Version

## ✅ Force SMBv1 (NT1)

```bash
smbclient -L 192.168.2.207 --option='client min protocol=NT1'
```

⚠️ SMBv1 is deprecated and insecure.

# 🟣 2️⃣ impacket-smbclient

## 🔹 What is impacket-smbclient?

Part of the Impacket toolkit used for advanced SMB interaction and authentication.

---

# 🔹 Help Menu

```bash
impacket-smbclient -h
```

---

# 🔹 Connection Methods

## ✅ Connect Using Username & Password

```bash
impacket-smbclient administrator:Anish@123@192.168.2.207
```

---

## ✅ Connect Using Domain Credentials

```bash
impacket-smbclient armour.com/administrator:Anish@123@192.168.2.207
```

# 🟣 3️⃣ smbget

## 🔹 What is smbget?

`smbget` is used to download files from SMB shares similar to `wget`.

---

# 🔹 Examples

## ✅ Download a File

```bash
smbget smb://192.168.2.207/data/file.exe
```

---

## ✅ Download with Authentication

```bash
smbget -U administrator smb://192.168.2.207/data/file.zip
```

---

## ✅ Save File to Specific Location

```bash
smbget -U administrator smb://192.168.2.207/data/file.zip -o /tmp/file.zip
```

---

# 🟣 4️⃣ smbtar

## 🔹 What is smbtar?

`smbtar` is used to archive SMB shares into tar files.

---

# 🔹 Examples

## ✅ Create Archive from Share

```bash
smbtar -s 192.168.2.207 -x data -t data.tar -v
```

---

## ✅ Create Archive with Authentication

```bash
smbtar -s 192.168.2.207 -x C$ -t data.tar -u administrator -p password -v
```

---

# 🟣 Important Notes

## ✅ smbclient Features

- Works similar to an FTP client
- Used for SMB enumeration and file transfer
- Supports authentication and anonymous access

---

## ✅ Administrative Shares

Examples:

- `C$`
- `ADMIN$`

👉 Require elevated or administrator privileges.

---

## ✅ Pass-the-Hash

- Requires valid NTLM hash
- Useful when password is unavailable

---

## ✅ SMBv1 Warning

⚠️ SMBv1 is outdated and insecure.

- Vulnerable to attacks like EternalBlue
- Should be disabled in modern systems

---

## ✅ Anonymous Access

- Possible when Null Sessions are enabled
- Often misconfigured in vulnerable systems
