# 📘 MySQL Nmap NSE Scripts

Nmap provides multiple **NSE (Nmap Scripting Engine)** scripts for MySQL enumeration, authentication testing, security auditing, and vulnerability assessment.

---

# 🟢 1. Check for Empty Passwords

Detect MySQL accounts configured with blank passwords.

```bash
nmap -v -sT -sV -p3306 \
--script=mysql-empty-password.nse \
192.168.10.103
```

### Example Output

```text
3306/tcp open mysql

| mysql-empty-password:
|   Accounts with empty passwords:
|     root
|     admin
|_
```

---

# 🟢 2. Enumerate MySQL Users

## 🟩 Without Authentication

Attempt anonymous user enumeration.

```bash

--script=mysql-users.nse \

```

---

## 🟩 With Authentication

```bash
nmap -v -sV -p3306 \
--script=mysql-users.nse \
```

```bash

```

```text

|   root
|   admin
|   developer
|   backup
|_
```

# 🟢 5. Execute SQL Queries

The `mysql-query` NSE script allows execution of SQL queries using valid credentials.

## 🟩 Show Databases

```bash
nmap -v -sT -sV -p3306 \
--script=mysql-query.nse \
--script-args='query="SHOW DATABASES",username=root,password=root' \
192.168.10.103
```

---

## 🟩 MySQL Version

```bash

--script=mysql-query.nse \
--script-args='query="SELECT VERSION()",username=root,password=root' \
```

# 🟢 15. Common Nmap Options

| Option | Description |
|--------|-------------|
| `-sT` | TCP Connect Scan |
| `-sV` | Service & Version Detection |
| `-Pn` | Skip Host Discovery |
| `-p3306` | Scan MySQL Port |
| `-v`, `-vv` | Verbose Output |
| `--script` | Execute NSE Script |
| `--script-args` | Pass Arguments to NSE Scripts |

# 🟢 19. Key Findings

- MySQL commonly listens on **TCP port 3306**.
- `mysql-info` retrieves version, handshake, and protocol information.
- `mysql-users` enumerates database accounts.
- `mysql-databases` lists accessible databases.
- `mysql-query` executes SQL queries using valid credentials.
- `mysql-variables` dumps server configuration.
- `mysql-dump-hashes` retrieves password hashes when privileges allow.
- `mysql-empty-password` detects blank passwords.
- `mysql-audit` performs security auditing.
- `mysql-vuln-cve2012-2122` checks for the historic authentication bypass vulnerability.
- Combining multiple NSE scripts provides comprehensive MySQL enumeration in a single scan.
