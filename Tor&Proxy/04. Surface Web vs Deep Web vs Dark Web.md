# Surface Web vs Deep Web vs Dark Web

---

## Overview

The internet is divided into three main layers:

- Surface Web (Clearnet)
- Deep Web
- Dark Web

---

# 1. Surface Web (Clearnet)

The **Surface Web** is the part of the internet that is publicly accessible and indexed by search engines.

## ✅ Examples

- Google, YouTube, Facebook  
- News websites, blogs, forums  

## 🔍 Key Characteristics

- Indexed by search engines  
- Easily accessible via browsers (Chrome, Firefox)  
- Represents only **5–10% of the internet**  

👉 Anything you can search on Google is part of the Surface Web.

---

# 2. Deep Web

The **Deep Web** includes all content that is **not indexed by search engines**.

## 📌 Examples

- Email accounts (Gmail, Outlook)  
- Online banking portals  
- Private databases  
- Subscription-based content (Netflix, academic journals)  

## 🔍 Key Characteristics

- Requires login or special access  
- Not accessible via search engines  
- Makes up **~90%+ of the internet**  
- Completely legal and widely used  

👉 Deep Web = Private content, not hidden for anonymity.

---

# 3. Dark Web

The **Dark Web** is a small portion of the Deep Web that requires special tools to access.

## 🛠 Access Tools

- Tor Browser  

## ✅ Examples

- `.onion` websites  
- Anonymous forums  
- Privacy-focused platforms  

## ⚠️ Potential Risks

- Illegal marketplaces  
- Hacking communities  
- Data leaks  

## 🔍 Key Characteristics

- Not indexed by search engines  
- Requires anonymity tools (e.g., Tor)  
- Focused on privacy and anonymity  

👉 Dark Web = Intentionally hidden and anonymous networks.

---

# 🧊 Internet Iceberg Analogy

- **Surface Web** → Tip of the iceberg (visible)  
- **Deep Web** → Large underwater portion (hidden but normal)  
- **Dark Web** → Small hidden section (anonymous)  

---

# ⚖️ Comparison Table

| Feature        | Surface Web | Deep Web   | Dark Web        |
|----------------|------------|------------|-----------------|
| Indexed        | ✅ Yes      | ❌ No       | ❌ No            |
| Accessibility  | ✅ Easy     | ⚠️ Restricted | ❌ Requires tools |
| Tools Needed   | ❌ No       | ❌ No       | ✅ Yes (Tor)     |
| Content Type   | Public     | Private    | Anonymous       |
| Legality       | ✅ Legal    | ✅ Legal    | ⚠️ Mixed         |

---

# ⚠️ Important Notes

- Deep Web is **not illegal**  
- Dark Web is **not entirely illegal**, but contains risky areas  
- Always prioritize **security and legality** when exploring
