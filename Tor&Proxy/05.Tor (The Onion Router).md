# Tor (The Onion Router)

Tor is a privacy-focused network that allows anonymous browsing by routing traffic through multiple encrypted layers.

---

## 🔁 How Tor Works

Instead of connecting directly to a website, Tor routes your traffic through multiple nodes:

1. Entry Node → Knows your IP but not destination  
2. Middle Relay → Passes encrypted traffic  
3. Exit Node → Connects to destination but doesn't know you  

👉 Each layer is encrypted (like an onion)

---

## 🔑 Key Features

- Anonymity → Hides real IP  
- Encryption → Multi-layer encryption  
- Decentralized → Volunteer-run network  
- Hidden Services → Access `.onion` sites  

---

## 🌐 Tor vs Normal Internet

| Feature        | Normal Browser | Tor |
|----------------|---------------|-----|
| IP Visibility  | Visible       | Hidden |
| Speed          | Fast          | Slow |
| Privacy        | Low           | High |
| Tracking       | Easy          | Difficult |

---

## 🛠 How to Use Tor

- Download Tor Browser  
- Install and connect  
- Start anonymous browsing  

---

## ⚠️ Important Safety Tips

- Do NOT log into personal accounts  
- Avoid downloading unknown files  
- Do NOT use normal browser alongside Tor for sensitive tasks  
- Prefer HTTPS websites  

---

## ❌ Limitations

- Slower speed  
- Some websites block Tor users  
- Exit nodes can see unencrypted traffic  
- Not 100% anonymous if misused  

---

## 🧾 Summary

- Tor = Anonymous browsing network  
- Uses encryption + relays  
- Useful for privacy, censorship bypass, hidden services

# 🏛 History of Tor

## Origins (1990s)

- Developed by researchers at **US Naval Research Laboratory**

### Key Contributors

- Paul Syverson  
- Michael G. Reed  
- David Goldschlag  

---

## 🎯 Goal

- Protect government communications  
- Enable anonymous intelligence gathering  

👉 This is where **Onion Routing** was invented

---

## 🚀 Public Release (Early 2000s)

- 2002 → Open-source release  
- 2004 → Public expansion  

### Why Public?

- More users = better anonymity  
- Larger network = harder tracking  

---

## 🧩 Tor Project Formation (2006)

- The Tor Project was established  

### Key Figure

- Roger Dingledine  

### Purpose

- Maintain and develop Tor  
- Promote privacy & freedom  

---

## 📈 Growth & Adoption

Used by:

- Journalists  
- Whistleblowers  
- Security researchers  
- Users in censored regions  

---

## ⚠️ Controversy & Misuse

Tor supports privacy but is also misused.

### Example

- Silk Road (dark web marketplace, shut down in 2013)

### Impact

- Increased law enforcement monitoring  
- Privacy vs security debates  

---

## 🔐 Modern Tor

- Maintained by Tor Project  
- Used for:
  - Privacy protection  
  - Censorship bypass  
  - Anonymous communication  

---

## 🕒 Timeline Summary

- 1990s → Onion routing developed  
- 2002 → Tor released  
- 2004 → Public adoption grows  
- 2006 → Tor Project founded  
- 2010s+ → Widespread usage

## ⚙️ Tor Setup (Linux)

    # Install Tor
    apt install tor

    # Start / Restart service
    systemctl restart tor

    # Check listening ports
    netstat -nltup

    # Filter Tor process
    netstat -nltup | grep tor

    # Check service status
    systemctl status tor

Expected:

    127.0.0.1:9050 → Tor SOCKS Proxy
